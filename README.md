## 🛠 三步快速跑起来（小白友好版）
### 第一步：安装依赖（复制粘贴命令就行💻）
先打开电脑的**终端/命令行**，依次输入以下命令，按回车执行～
> 没装`uv`的宝子先装uv：
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```
> 装好uv后，继续执行：
```bash
# 进入项目文件夹（把路径换成你自己的项目地址哦）
cd /你的项目路径/notion-auto-to-xiaohongshu-images
# 创建虚拟环境
uv venv .venv
# 激活环境（Mac/Linux）
source .venv/bin/activate
# 安装项目所有依赖
uv pip install -r requirements.txt
# 安装chromium（自动适配，直接执行）
uv run playwright install chromium
```

### 第二步：启动神器✨
终端里继续输入命令，回车启动：
```bash
python web_app.py
```
启动成功后，打开浏览器，输入下面的地址回车，就能看到操作页面啦👇
```
http://127.0.0.1:8123
```

### 第三步：Notion导出准备（超关键📝）
先把Notion里的内容导出成指定格式，避免缺图/显示不全：
1. 打开Notion目标页面，点右上角「···」
2. 选择「Export」（导出）
3. 格式**必选HTML** 📄
4. 导出后会得到zip文件夹（这步最简单）

## 🎨 一键生成小红书配图（操作超丝滑）
打开`http://127.0.0.1:8123`后，四步搞定配图👇
### 1️⃣ 上传文件
点击上传，选**刚才打包的Notion zip文件**（推荐），也可以传单个html（可能缺图哦）
### 2️⃣ 选择参数（小白默认就行✅）
- 导出图片选项：选想要的尺寸（1440×2400满屏超推荐）+ 清晰度（默认就高清）
- 切分策略：小白直接用默认，不用调！
### 3️⃣ 预览效果
点击「生成全文预览」，中间区域上下滑动，看看分页效果，不满意再微调参数～
### 4️⃣ 高清导出
点击「开始高清导出」，右侧看任务进度，等几秒就能得到小红书专属高清配图啦🎉

## ❗ 小白避坑小技巧（必看💡）
1. 启动后打不开页面？👉 本地浏览器里输入`http://127.0.0.1:8123`或`http://localhost:8123`，别用0.0.0.0
2. 图片显示不全/模糊？👉 提高「导出清晰度倍率」+「锐化强度」
3. 内容切分不好看？切分到了文字或图片？👉 调整切分策略里的「最小分段高度」这个参数，小白建议直接默认～
